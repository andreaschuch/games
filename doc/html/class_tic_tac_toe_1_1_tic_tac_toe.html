Checks if a player has won the game, and returns the fields that made the victory.

A player has won if he has placed his marks in a line covering the board from one side to the other. The line can be horizontal, vertical or diagonal. Note: Will not retrieve multiple lines.

Returns
The list of winning fields that make up the line.
