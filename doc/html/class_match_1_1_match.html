Detailed Description

Class for playing a match of a game.

The following functionalities are supported

  1. Playing a game manually using different graphical interfaces.
  2. Having machine players competing against one another.
  3. Manually playing against a machine player.
    Author
    Andrea Schuch

def Match.Match.is_game_over (   self)

Checks if the game has ended.

A games is over if

  • either of the player has won
  • or there is a draw preventing either playing from winning. Sets winner variable to current_player(), if not a draw.
    Returns
    True if the game is over, and false otherwise.

def Match.Match.play (   self)

Starts a new match and runs the game until the game-over conditions are met.

Machine players are asked to provide a move using Player.next_move(). Human players' moves are collected via the gui using Gui.next_move().
