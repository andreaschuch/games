Looking at the board, the learner adds missing moves to his internal history.

Should be called after every end-of-game.

Parameters
gameThe game.
rewardThe reward that should associated with the current state of the game (e.g. 100 if the learner is winning, -100 if the learner is loosing, 0 for a draw).
last_playerThe player who made the last move in the game.
